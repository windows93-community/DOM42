DOM42 documentation

Introduction

 DOM42 is a Windows93 library made to simplify designing GUI. In the start you need to create a window and load it as a DOM42 object (you can either use  window({..}) and new DOM42(win) or you can use $window42({...})) After the window is created and loaded, you need to create widgets and add then to your window.

Example

/* load and create a new window */
var win = $window42({title: "Example window"});
/* Export widgets from $widget */
$widget.Export()
/* Create a new label */
var lab = Label("Click the button to say hello!");
la1.setPosition(50, 50);
/* Create a new button */
var btn = Button("Click me!");
btn.setPosition(100, 80);
/* Add function on click to the button */
btn.onClick(function() { console.log("Hello!") });
/* Add widgets to the window */
win.add(lab);
win.add(btn);

Widgets

All Widget classes are saved in $widget. To avoid typing $widget a lot, there is $widget.Export(), which will make all widgets global. $widget also contains a function positionBy(text), where you can change between pixels ("pixel") or size relative to window size ("relative", helpfull when windows are resizable)

  • Button(text)

    • onClick(function)
    • setStyle(object)
    • setPosition(x, y, w, h)

  • Label(text, size)

    • setStyle(object)
    • setPosition(x, y, w, h)

  • Entry(size)

    • setText(text)
    • getText(text)
    • setStyle(object)
    • setPosition(x, y, w, h)

  • Input(type, size)

    • setText(text)
    • getText(text)
    • setStyle(object)
    • setPosition(x, y, w, h)

  • Section()

    • setStyle(object)
    • setPosition(x, y, w, h)

DOM42 objects

DOM42 objects can load windows and section widgets. When DOM42 object is created, we can either add widgets to it (by using .add(widget)) or remove widgets from it (by using .remove(widget)).
